# Changelog бизнес-правил

Фиксируются только изменения бизнес-правил и поведения модулей, не технические правки кода (для них есть git-история).

## 2026-07-08

- Конструктор сценариев зафиксирован как визуальный редактор схем: сценарий — граф из блоков и связей на холсте, а не линейный список шагов. Добавлены правила: единственный блок «Старт»; выходы блока — по одному на кнопку/элемент, «Продолжить» и «Любая другая фраза» (нераспознанный ввод; без связи — повтор текущего шага); правки применяются только после публикации сценария с валидацией лимитов WhatsApp и подключенности выходов. Создано описание модуля [modules/bot-constructor.md](modules/bot-constructor.md).

## 2026-07-07

- Документация инициализирована: ТЗ перенесено из Google Docs в [technical-specification.md](technical-specification.md), извлечена сводка [business-rules.md](business-rules.md). Блок «Обновление для ТЗ» (поле `expires_at`, Utility Templates, автоматический 30-дневный цикл актуальности) интегрирован в соответствующие модули.
